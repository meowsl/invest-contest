Система анализа Целей Устойчивого Развития

Что такое ЦУР?

Это набор из 17 взаимосвязанных целей, разработанных в 2015 году Генеральной ассамблеей ООН в качестве «плана достижения лучшего и более устойчивого будущего для всех».

Отслеживая статистические данные показателей государство способно обеспечивать наиболее качественные и благоприятные изменения для того или иного региона РФ.

Этап 1-2

Проведя тщательный анализ имеющихся данных наша команда пришла к выводу, что для анализа временного ряда критически мало данных. Следуя из этого каждый участник приступил к поиску дополнительных сведений для расширения, а следовательно для конкретизации имеющейся статистики.
